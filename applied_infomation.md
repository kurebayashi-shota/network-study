# 技術

- アサーションチェック:bool 値が正しいか確認する
- コード追跡(トレーサー):関数の実行結果を残してくれる
- テストガバレッジ分析(モニター):どこまでテスト出来ているか

## CPU

- スーパーパイプライン:パイプラインを細分化
- スーパースカラ:パイプラインを複数用意
- VLIW:命令をベリーロングにして負荷軽減
- ベクトル処理方式:ベクトルがどうのこうの

## ネットワーク

- ATM:分割して送信するので LAN が高速化

# 管理

- RASIS:信頼・可用・保守・完全・機密
- CVSSv3:脆弱性評価システム(基本評価基準、現状評価基準、環境評価基準)
- MDM:モバイル管理する

# 稼働率

| RTO  | RPO  | 　　 MTTR      | 　 MTBF  |
| ---- | ---- | -------------- | -------- |
| 時間 | 時点 | 復旧までの時間 | 稼働時間 |

# 経営

- BI:膨大なデータを収集・分析・可視化して経営・意思決定を支援する仕組み
- BPR:リエンジニアリング。既存を抜本的に見直し、再構築する
- BPM:ビジネスプロセスを可視化・分析・改善して永続的に最適化
- BPO:アウトソーシング,オフショアは海外
- EA:BA,DA,AA,TA を全体視点で整理・設計-ビジネス、データ、アプリ、技術

# 考え方

- SOA:サービス単位で分割

# 生産管理

- MRP:所要量計算
- CRP:計算する。必要なら山崩しで対応
- BOM:PEMSS。E と M が重要で M は E を基に作られる
